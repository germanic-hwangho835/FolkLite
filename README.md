## 🧭 What FolkLite is

FolkLite is the MIUIX branch of the FolkPatch project. It is made for users who want a simple way to apply FolkPatch updates on Windows without dealing with code or setup tools.

This README focuses on the basic steps to get FolkLite downloaded, opened, and used on a Windows PC. If you only want the fastest path, go to the download section and get the latest release.

## ✨ What it does

FolkLite helps you:

- download the MIUIX build from one place
- run the app on Windows with a few clicks
- keep the patching flow simple
- use a version made for the FolkPatch line
- avoid manual file handling where possible

It is built for end users, so the steps stay short and plain.

## 💻 What you need

Before you start, make sure your PC meets these basic needs:

- Windows 10 or Windows 11
- 2 GB of free disk space

For best results:

- keep Windows up to date

## 🪟 Install or set up on Windows

FolkLite is meant to be easy to start on Windows. After the file downloads:

1. open File Explorer
2. go to your Downloads folder
3. find the FolkLite file you downloaded
4. if it is a ZIP file, right-click it and choose Extract All
5. pick a folder you can find later, such as Desktop or Documents
6. open the extracted FolkLite folder
7. look for the main app file, such as an `.exe` file
8. double-click it to start

If Windows shows a security prompt, choose the option that lets you open the file.

If the app needs files in the same folder, keep them together. Do not move one file by itself unless the release page says you can.

## 🔧 Common problems

### The app does not open

Try this:

1. right-click the app file
2. choose Run as administrator
3. check whether Windows blocked the file
4. move the folder to Desktop or Documents
5. try again

### Windows says the file is unsafe

This can happen with downloaded tools. If the file came from the official release page, check that you downloaded the latest release and not a copy from another site.

### The app closes right away

Try:

1. extract the ZIP file again
2. make sure all files stayed together
3. run the app from the extracted folder
4. avoid spaces or symbols in the folder name
5. use a folder path that is short, such as `C:\FolkLite`

### The patch fails

Try:

1. check that the source file is complete
2. make sure you picked the right MIUIX build
3. close other tools that may use the same file
4. free up disk space
5. try the process again
